💸 Smart Expense Splitter

A Python-based command-line application that helps split expenses among a group of people and calculates who owes whom, similar to Splitwise.

---

🚀 Features

- 👥 Add multiple users
- 💰 Record expenses for each person
- ⚖️ Equal expense splitting
- 📊 Automatic balance calculation
- 🔄 Smart settlement (who pays whom)
- ❌ Handles invalid inputs
- 📌 Supports multiple payers
- 🧠 Uses efficient algorithms for fair distribution

---

🧠 How It Works

1. Users are added to the system
2. Expenses are recorded per person
3. Total expense is calculated
4. Expense is equally divided among all users
5. Each person's balance is calculated:
   - Positive → should receive money
   - Negative → should pay money
6. Final settlements are generated:
   - Shows exactly who pays whom and how much
